# VisionClass AI

A male/female gender prediction application that uses a camera or gallery image based on a deep learning model; with some datasets for testing.


## Installation

```bash
pip install -r requirements.txt
```

## use

From the project file :
   ```bash
   cd "code&dataset"
   streamlit run app.py
   ```

## Project structure

- `app.py` : Streamlit application
- `code.ipynb` : model training notebook
- `modele_homme_femme.h5` : pre-trained model
